# --- Definição de um AFD que aceita cadeias terminadas em "01" ---

# Q - Conjunto de estados
ESTADOS: {q0,q1,q2}

# Σ - Alfabeto
ALFABETO: {0,1}

# q₀ - Estado Inicial
INICIAL: {q0}

# F - Conjunto de Estados Finais
FINAIS: {q2}

# δ - Função de Transição
# Formato: estado_atual, simbolo_lido -> proximo_estado
TRANSICOES:
q0, 0 -> q1
q0, 1 -> q0
q1, 0 -> q1
q1, 1 -> q2
q2, 0 -> q1
q2, 1 -> q0


exemplo de cadeia para umm ADF para um alfabeto (0,1):
>1101
